GetUnsequencedHashCode()

The unordered collection hashcode is defined as the sum of h(hashcode(item)) over the items of the collection, where the function h is a function from int to int of the form t -> (a0t+b0)^(a1t+b1)^(a2*t+b2), where the ax and bx are the same for all collection classes.

The current implementation uses fixed values for the ax and bx, specified as constants in the code.

public abstract Int32 GetUnsequencedHashCode()

The unordered hashcode of this collection.
