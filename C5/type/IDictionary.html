public interface IDictionary<K, V> : ICollectionValue where K : object where V : object

Implements

  • C5.ICollectionValue>

Summary

A dictionary with keys of type K and values of type V. Equivalent to a finite partial map from K to V.

Used by

  • GuardedDictionary
