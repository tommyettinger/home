public interface ICollectionValue<T> : IShowable, IEnumerable where T : object

Implements

  • C5.IShowable
  • System.Collections.Generic.IEnumerable

Summary

A generic collection that may be enumerated and can answer efficiently how many items it contains. Like IEnumerable, this interface does not prescribe any operations to initialize or update the collection. The main usage for this interface is to be the return type of query operations on generic collection.
