Summary

A generic collection to which one may add items. This is just the intersection of the main stream generic collection interfaces and the priority queue interface, ICollection and IPriorityQueue.

Used by

  • GuardedCollection

Add(T item)

Add an item to this collection if possible. If this collection has set semantics, the item will be added if not already in the collection. If bag semantics, the item will always be added.
