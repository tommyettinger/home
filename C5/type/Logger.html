Log

Gets or sets the log.

public Action<String> Log { get; set; }

Remarks

If Log is not set it will return a dummy action x => { return; }) eliminating the need for null-reference checks.

Example

The following is an example of assigning a observer to the logging module: Logger.Log = x => Console.WriteLine(x);
