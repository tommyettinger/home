Syntax

public class WrappedArray<T> : IList, IList where T : object

Implements

  • C5.IList
  • System.Collections.Generic.IList

Summary

An advanced interface to operations on an array. The array is viewed as an IList of fixed size, and so all operations that would change the size of the array will be invalid (and throw FixedSizeCollectionException
